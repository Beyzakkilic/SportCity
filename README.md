- Flutter SDK (sürüm 3.24.5)
- Dart SDK (sürüm 3.5.4)

### Kurulum Adımları
1. Projeyi bilgisayarınıza indirin:
```bash
git clone [https://github.com/Ysnkrgzz/SportCity.git]
```
2. Proje Klasörüne gidin: 
```bash
cd [SportCity]
```
3. Gerekli paketleri yükleyin
```bash
flutter pub get
```
4. Uygulamayı Başlatın
```bash
flutter run
```
